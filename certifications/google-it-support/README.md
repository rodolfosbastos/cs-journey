### Course 2 — The Bits and Bytes of Computer Networking

**Module 1:** Introduction to networking — what a network is, basic terminology, the TCP/IP model.

**Module 2:** The Network Layer — IP addressing, subnetting, and routing.

- **IP addressing:** Every device on a network has an IP address. IPv4 uses 32-bit addresses written in dotted-decimal notation (e.g. 192.168.1.1).
- **Subnetting:** Divides a network into smaller sub-networks. A subnet mask defines which part of an IP address is the network portion and which is the host portion. CIDR notation (e.g. /24) expresses this compactly.
- **Routing:** The process of forwarding packets between networks. Routers use routing tables to determine the best path. Key concepts: default gateway, static vs. dynamic routing.
